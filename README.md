🏨 Hotel Revenue Analytics — Power BI Dashboard
A comprehensive Power BI dashboard built to explore hotel performance metrics including revenue trends, occupancy patterns, and booking behavior. Designed with a focus on clarity and interactivity for hospitality stakeholders.

📊 What This Project Covers
The report dives into:

Revenue tracking and historical trends
Occupancy rate monitoring
Room category and booking analysis
Week-over-week revenue comparisons
Location-wise performance breakdown

Dynamic slicers allow users to filter and explore data on their own terms.

🛠️ Tech Stack

Power BI Desktop
DAX (Data Analysis Expressions)
Custom Tooltips
Conditional Formatting
Card & Table Visuals


📁 Project Structure

Hospitality Domain.pbix — Core Power BI report file
images/dashboard-screenshot.png — Preview of the dashboard
README.md — Project documentation


🚀 Getting Started

Clone or download this repository
Open Hospitality Domain.pbix in Power BI Desktop
Use the slicers and filters to explore insights interactively


✨ Key Highlights

Dynamic filtering for customized data views
KPI cards with descriptive tooltips
Conditional formatting with icon indicators
Clean, intuitive visual layout
